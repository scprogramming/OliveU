Getting Started with Python

Author: Scott Cosentino

Python is a high-level interpreted programming language. Let’s break down this definition to understand better what that means. A programming language is a language that can be used to write programs for a computer. A high-level language lets programmers write programs without needing to worry about the details of how the computer runs and stores the program. This abstraction of detail makes it easier to write programs but sacrifices some efficiency. Most programmers work in high-level languages. However, some domains, like system programming, use more low-level languages. With any programming language, your code must be translated into machine code so your CPU can understand and execute the code. There are two main approaches to this translation. The first approach is called compiling. With compiling, all the code associated with a program is translated into machine code. This creates an executable file that can be run on the target computer. The second approach is called interpretation. With interpreted languages like Python, program code is run directly and translated into machine code as it runs.

To be able to write Python code, the first thing you need to do is download an interpreter. You can download Python from here. You will be able to download an installer for your operating system and install it as you would any other program. Python installs various components, but we are mainly interested in two components: the Python interpreter and IDLE.

Python Interactive Mode

You can access Python in interactive mode by typing python into your computer command line. You will see an output like the following image.

Interactive mode allows you to execute code one line at a time by typing the code into the command line. For example, type the line print(“Hello!”). You will see the text Hello printed on the screen.

The interactive mode of Python is ideal for situations where you want to test some code to see the result quickly. The Python command can also be used to run code that is stored in a file. To see this, exit interactive mode by typing quit().

Running Code in Files

To see how Python can run code in a file, create a file somewhere on your computer. I will create my file in C:\source\test\ and name the file test.py. The .py extension is the typical file extension used for Python programs. Inside the file, write the code print(“Hello!”). Save the file, then open your command line. To run your code, run python yourFileNameHere. For example, my file is named test.py, so I use the command python test.py. You will see the code result in your command line.

This method of running code is ideal for larger projects and projects that need to be run multiple times. Writing code in text files is easy enough when you have a small program, but it is helpful to have more powerful tools for bigger projects. We often use an IDE, or integrated development environment, for writing programs. An IDE provides tools for writing and debugging code, making it easier to write working programs. The best IDE to use is really a subjective choice. Python comes with an IDE named IDLE, which provides some basic IDE functionality. I prefer to use PyCharm for my personal projects, which I will primarily demonstrate with. To download PyCharm, navigate to the JetBrains website. This site will provide an installer for each operating system and instructions for installing PyCharm.

Basics of PyCharm

When launching PyCharm, you will be prompted to open or create a project. Select New Project to create a Python project.

When you create a new project, you will be asked for a location to store the project. Select any location you want to use for saving code on your computer. The second section will ask you to create a virtual environment for Python. PyCharm creates a unique Python interpreter for each project you create. This allows you to change configurations for your project without affecting other programs that use Python on your computer. The base interpreter should point to your python.exe file. The following image shows a sample project setup for PyCharm.

When you create your project, you will be redirected to the main project window. On the left, you will see all of the files in your project.

To create a new file in your project, right click on the top folder, then select new. For example, I would right-click the test folder to create a new file. You will be able to create new Python files and other files as needed. To open a file, double-click on it in the left project sidebar. When you do this, the file contents will show in the middle window of PyCharm.

To run the program, select the green play button in the top right of PyCharm. The dropdown next to the play button lets you select which file you want to run. By default, this is set to the current file, which will run the file currently open in the main window.
