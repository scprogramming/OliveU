Most programs you write will take some form of input, perform an operation, then provide an output. We've learned the process of outputting data using the print function. This tutorial will explore the process of taking user input using the input function.

The Basics of the Input Function

You will generally use the input function in the form of an assignment statement, as shown in the following code:

userInput = input("Enter some input")

Let's break down this code to understand better exactly how it works. The input function takes in one argument. This argument is the prompt that will be displayed to the user when the application prompts them for input. In this example, the prompt is Enter some input. When the user provides input, Python will read the input and store it in the variable named userInput. To see this code in action, let's add a print to see what Python stores in the userInput variable. 

userInput = input("Enter some input")
print(userInput)
