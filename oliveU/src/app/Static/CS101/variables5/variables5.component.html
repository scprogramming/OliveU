Variables in Python

CS101 - Introduction to Programming with Python
Author: Scott Cosentino

In programming, programs use variables to store data in memory. You can access data in a variable and change the data as the program runs. Programmers usually use variables to keep track of inputs and outputs. For example, consider a program that takes the total of a sale as an input and calculates the sales tax as the output. In this example, the total of the sale needs to be stored in a variable so it can be used in the sales tax calculation. Let's take a look at how we can declare variables in Python.

Declaring variables in Python

In Python, we create a variable using an assignment statement. The following code shows an example of a few assignment statements.

num = 15
my_text = "Hello!"
decimalData = 3.14

From this example, you can see the general format of the assignment statement. TThe left side of the equals sign is the variable's name, and the right-hand side is the variable's value. The equals sign is called in assignment operator in this statement. In this example, we defined a variable named num with a value of 15, a variable named my_text with a value of Hello!, and a variable decimalData with a value of 3.14.

Regarding the naming of variables, there are a few rules that you should be aware of:

  • You cannot use Python keywords as variable names (for example, print is not a valid variable name)
  • A variable name cannot have spaces in it
  • The first character of the name must be either a letter or underscore character
  • After the first character, you can use letters, numbers, or underscores
  • Variable names are case sensitive

It is possible to use these variables in your application code. For example, you can print a variable's value by placing the variable in a print function.

num = 15
print(num)

When this code runs, Python will attempt to determine the value of the num variable. To do this, Python goes to the memory location associated with the variable. In this memory location, Python will find the value 15. Python will then print the value 15 on the screen.

Variable Reassignment

So far, you've learned how to create a variable through assignment and use a variable in a function like print. In this section, we will see how variable values can change. To start, it is possible to assign a new value to a variable after it is created. The code below demonstrates this process. 

num = 15
num = 20
print(num)

In this example, the variable num is initially set to the value 15. The next line updates the value to 20. When the print funciton runs, the last value the variable had, 20, is print to the screen.

Changing a variables value is often done using some form of operation. For example, we can add a number to a variable to change its value.

num = 15
num = num + 1
print(num)

In this example, num originally has a value of 15. The next line takes the value of num, adds 1 to it, then stores the result back in the num variable. When the print occurs, the value of num is now 16.
